# matomo-metabase

Extract matomo data and push to Postgres

## Deployment

| name              | value                      |
| ----------------- | -------------------------- |
| START_DATE        | 2021-06-01                 |
| DESTINATION_TABLE | matomo                     |
| MATOMO_KEY        | matomo api token           |
| MATOMO_SITEID     | matomo site id             |
| MATOMO_URL        | matomo url                 |
| PGDATABASE        | Postgres connection string |
